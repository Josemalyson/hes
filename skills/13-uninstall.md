# HES Skill — 13: Uninstall

> **Trigger:** `/hes uninstall` | `/hes remove` | `/hes clean`
> **Objective:** Remove everything HES created, leaving the project in its pre-HES state.
> **Safety contract:** requires explicit double confirmation before any deletion.
> **Principle:** the harness must be removable cleanly — no residue, no side effects.

---

## ◈ STEP 1 — INVENTORY WHAT WILL BE REMOVED

Before touching anything, build the full removal manifest:

```bash
bash scripts/hooks/log-action.sh LLM_DECISION STARTED \
  "uninstall-manifest" "building removal inventory"

# --- HES-owned directories ---
HES_DIRS=(
  ".hes/"                   # harness state, specs, decisions, tasks, schemas, agents
)

# --- HES-installed root files ---
HES_ROOT_FILES=(
  "SKILL.md"
  "AGENTS.md"              # if generated by HES bootstrap
  "ARCHITECTURE.md"        # if generated by HES bootstrap
  "CONTRIBUTING.md"        # if generated by HES bootstrap
  "INSTALL.md"
  "CHANGELOG.md"
  "security-policy.yml"
  "setup"
)

# --- IDE config files generated by HES ---
HES_IDE_FILES=(
  ".claude/CLAUDE.md"          # Claude Code
  ".claude/commands/"          # Claude Code custom commands
  ".cursor/skills/"            # Cursor
  ".cursor/rules/hes.mdc"      # Cursor
  ".kiro/skills/"              # Kiro
  ".kiro/steering/hes.md"      # Kiro
  ".agents/skills/"            # OpenHands / Codex
  ".github/copilot-instructions.md"  # Copilot
  ".github/workflows/harness-validation.yml"
  ".windsurfrules"             # Windsurf
  ".gemini/"                   # Gemini CLI (if HES-generated)
)

# --- skills/ directory ---
HES_SKILLS_DIR="skills/"

# --- git hooks (if installed as real hooks) ---
HES_HOOKS=(
  ".git/hooks/pre-commit.hes"
  ".git/hooks/commit-msg.hes"
)
```

## ◈ STEP 4 — LOG FINAL EVENTS (before deletion)

Write the farewell event to events.log BEFORE removing it:

```bash
bash scripts/hooks/log-action.sh LLM_DECISION STARTED \
  "uninstall" "writing final events before removal"
```

Append to `.hes/state/events.log`:
```json
{
  "timestamp": "{{NOW}}",
  "session_id": "{{SESSION_ID}}",
  "feature": "global",
  "from": "HARNESS_INSTALLED",
  "to": "UNINSTALLED",
  "agent": "hes-uninstall",
  "metadata": {
    "project": "{{PROJECT_NAME}}",
    "harness_version": "{{HARNESS_VERSION}}",
    "completed_cycles": {{COMPLETED_CYCLES}},
    "total_features": {{TOTAL_FEATURES}},
    "uninstalled_at": "{{NOW}}",
    "operator": "user-confirmed"

```

Optionally export a final summary report before deletion:
```bash
# Export readable summary to project root (user keeps this if they want)
cat .hes/state/events.log > hes-history-export-{{DATE}}.jsonl 2>/dev/null || true

echo "📄 Exported: hes-history-export-{{DATE}}.jsonl + hes-lessons-export-{{DATE}}.md"
```

## ◈ STEP 5 — EXECUTE REMOVAL

```bash
bash scripts/hooks/log-action.sh EXEC_CMD STARTED "rm -rf .hes/ skills/" "removing harness"

# 1. Remove harness state and skill-files
rm -rf .hes/
rm -rf skills/
rm -f SKILL.md

# 2. Remove HES root files (only if HES-generated — check git blame or marker)
for f in AGENTS.md ARCHITECTURE.md INSTALL.md CHANGELOG.md CONTRIBUTING.md security-policy.yml setup; do
  [ -f "$f" ] && rm -f "$f" && echo "  removed: $f"
done

# 3. Remove IDE config files (only HES-generated)
rm -f  .claude/CLAUDE.md
rm -rf .claude/commands/
rm -f  .cursor/rules/hes.mdc
rm -rf .cursor/skills/
rm -f  .kiro/steering/hes.md
rm -rf .kiro/skills/
rm -f  .github/copilot-instructions.md
rm -f  .github/workflows/harness-validation.yml
rm -rf .agents/skills/
rm -f  .windsurfrules

# 4. Clean up empty IDE dirs if HES was the only content
for d in .claude .cursor .kiro .agents; do
  [ -d "$d" ] && [ -z "$(ls -A "$d")" ] && rmdir "$d" && echo "  removed empty: $d/"
done

# 5. Remove git hooks (if HES-installed)
rm -f .git/hooks/pre-commit.hes
rm -f .git/hooks/commit-msg.hes

# 6. Remove scripts/ if HES-generated
# (check: if scripts/ only contains hooks/ and ci/ from HES)
HES_SCRIPTS_ONLY=true
for f in $(find scripts/ -type f 2>/dev/null | grep -v -E "scripts/hooks/(log-action|step-budget|context-offload|telemetry)\.sh|scripts/ci/validate-harness\.py"); do
  HES_SCRIPTS_ONLY=false; break
done
$HES_SCRIPTS_ONLY && rm -rf scripts/ && echo "  removed: scripts/ (HES-generated only)"
```

## ◈ STEP 6 — VALIDATE CLEAN STATE

```bash
# Verify nothing HES-owned remains
REMAINING=""
[ -d ".hes" ]    && REMAINING="$REMAINING .hes/"
[ -d "skills" ]  && REMAINING="$REMAINING skills/"
[ -f "SKILL.md" ] && REMAINING="$REMAINING SKILL.md"

if [ -z "$REMAINING" ]; then
  echo "✅ Clean — no HES artifacts remaining"
else
  echo "⚠️  Still present (manual removal needed): $REMAINING"
fi
```
